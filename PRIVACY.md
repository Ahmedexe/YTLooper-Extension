# Privacy Policy for YouTube Interval Looper

Effective date: May 23, 2026

YouTube Interval Looper does not collect, store, sell, share, or transmit any personal information.

The extension works locally in your browser to help you select and repeat a specific interval of a YouTube video. It may interact with the currently open YouTube video page only to read or control the video timestamp for the purpose of looping the selected interval.

## Data Collection

This extension does not collect any personal data, browsing history, YouTube account information, video history, or user activity.

## Data Storage

The extension may store user-selected start and end timestamps locally in the browser, if needed for functionality. This data remains on the user’s device and is not sent to any server.

## Data Sharing

No data is shared with third parties.

## Remote Servers

This extension does not transmit user data to external servers.
